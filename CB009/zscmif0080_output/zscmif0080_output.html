Code listing for: ZSCMIF0080_OUTPUT

Description: 商品凭证输出消息（Output Message）产生Idoc
REPORT ZSCMIF0080_OUTPUT.

* 该程序根据商品凭证产生Idoc输出。
* 该程序可直接执行，也可配置在商品凭证输出控制中，通过输出消息触发产生Idoc

include ZSCMIF0080_OUTPUT_TOP.
include ZSCMIF0080_OUTPUT_SCR.
include ZSCMIF0080_OUTPUT_F01.


START-OF-SELECTION .
  DATA LV_RC TYPE SY-SUBRC .

  PERFORM ENTRY USING LV_RC 'X' .
  READ TABLE GT_RETURN INTO GS_RETURN INDEX 1 .
  IF SY-SUBRC = 0 .
    MESSAGE ID GS_RETURN-ID
            TYPE GS_RETURN-TYPE
            NUMBER GS_RETURN-NUMBER
            WITH GS_RETURN-MESSAGE_V1 GS_RETURN-MESSAGE_V2
                 GS_RETURN-MESSAGE_V3 GS_RETURN-MESSAGE_V4 .
  ENDIF.

*Text elements
*----------------------------------------------------------
* 001 输入要产生IDoc的商品凭证号


*Selection texts
*----------------------------------------------------------
* P_MBLNR D       .
* P_MJAHR D       .
