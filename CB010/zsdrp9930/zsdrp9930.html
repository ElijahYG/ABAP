*1.根据POS小票号查询
*  根据输入的小票号，在BKPF表中的BKTXT 字段中查找。把BELNR（会计凭证编号） 、GJAHR（会计年度）、AWTYP（参考交易）、AWKEY（字段参考关键）、BUKRS （公司代码）找出。
*  此处的会计凭证为成本会计凭证号。
*  商品凭证与会计凭证的关系 BKPF-AWTYP = 'MKPF'
*                         BKPF-AWKEY = 商品凭证号码（MKPF-MBLNR）+商品凭证年度（MJAHR）
*
*通过以上关系能找出相应的商品凭证号码及商品凭证年度。
*根据输入的小票号，在VBRK表中的BKTXT字段中查找。把VBELN （开票凭证 ）找出。
*根据找出的VBELN ,再去BKPF表中根据 AWKEY 等于刚找出的VBELN  且AWTYP = 'VBRK'   BELNR （会计凭证编号）。此处的 = 'VBRK'   反查出的BELNR （会计凭证）为收入会计凭证号。
*新增，扣点门店      MKPF   BKPF  VBRK
*      礼小票金卡99   无     无    有
*      不带99普通小票 有     无    有
* 扣点门店不生成BKPF不能按照以上逻辑去取
* 故如果在BKPF表中 未取到数时  先去MKPF表里取 再去VBRK取，如是99小票在MKPF表里取不到，则直接去VBRK里取数。

*2.根据会计凭证查询
*  同上

*3.根据商品凭证查询
* 通过输入的商品凭证及年度在MKPF表中找到，MBLNR（商品凭证） MJAHR（年度） BKTXT（小票号） 。
* 再根据找出的内表中的小票号，去BKPF表中找，后续取数逻辑与根据POS小票号查询 逻辑相同。
*新增，扣点门店      MKPF   BKPF  VBRK
*      礼小票金卡99   无     无    有
*      不带99普通小票 有     无    有
* 扣点门店不生成BKPF 及 99小票不生成MKPF。不能按照以上逻辑去取
* 故如果在MKPF表中未取到数，则直接去VBRK中取。
*   如果在MKPF表中 取到数 但未在BKPF表中取到数，可根据内表中的小票号去 VBRK表取。后续取数逻辑与根据POS小票号查询 逻辑相同。

*4.根据发票凭证查询
*根据输入的发票凭证在VBRK中找到VBELN（发票凭证号）、EX_POSNUM（小票号）、ERDAT（创建日期） 。
*再根据找出的内表中的小票号，去BKPF表中找，后续取数逻辑与根据POS小票号查询 逻辑相同。
*新增，扣点门店      MKPF   BKPF  VBRK
*      礼小票金卡99   无     无    有
*      不带99普通小票 有     无    有
* 扣点门店不生成BKPF 及 99小票不生成MKPF。不能按照以上逻辑去取
* 在没有生成BKPF时，取完VBRK 后去MKPF中去按小票号取数

*取数函数
FORM GET_BKPF.
  DATA LT_VBELN TYPE STANDARD TABLE OF STRU_0020.
  DATA WA_VBELN TYPE                   STRU_0020.
  DATA LT_TABLE1 LIKE IT_TABLE.
  DATA WA_TABLE1 LIKE WA_TABLE.
  DATA LT_TABLE2 LIKE IT_TABLE.
  DATA WA_TABLE2 LIKE WA_TABLE.
  DATA LT_TABLE3 LIKE IT_TABLE.
  DATA WA_TABLE3 LIKE WA_TABLE.
  DATA LT_TABLE4 LIKE IT_TABLE.
  DATA WA_TABLE4 LIKE WA_TABLE.
  DATA: ZBELNR  TYPE BKPF-BELNR,
        ZVBELN2 TYPE VBRK-VBELN.

  CLEAR: LT_TABLE1,WA_TABLE1,ZBELNR,ZVBELN2.

  SELECT B~BKTXT
         B~BELNR
         B~GJAHR
         B~AWTYP
         B~AWKEY
         B~BUKRS
    FROM BKPF AS B
    INTO CORRESPONDING FIELDS OF TABLE IT_TABLE
    WHERE B~BKTXT   IN S_POSNUM
      AND B~BELNR   IN S_BELNR
      AND B~GJAHR   IN S_GJAHR
      AND B~BELNR   IN S_BELNR.

  IF IT_TABLE IS NOT INITIAL.

    LT_TABLE3 = IT_TABLE .
    DELETE LT_TABLE3 WHERE BKTXT IS NOT INITIAL.
    DELETE IT_TABLE  WHERE BKTXT IS  INITIAL.
*如果输入的为收入会计凭证，那么BKTXT 无值。
    IF IT_TABLE IS NOT INITIAL.

      LOOP AT IT_TABLE  ASSIGNING <FS_TABLE> .
        IF <FS_TABLE>-AWTYP = 'MKPF' .
          <FS_TABLE>-MBLNR = <FS_TABLE>-AWKEY+0(10) .
          <FS_TABLE>-MJAHR = <FS_TABLE>-AWKEY+10(4).
        ENDIF.
      ENDLOOP.


      SELECT V~VBELN
             V~EX_POSNUM
             V~ERDAT
        FROM VBRK AS V
        INTO CORRESPONDING FIELDS OF TABLE LT_TABLE1
         FOR ALL ENTRIES IN IT_TABLE
        WHERE V~EX_POSNUM   = IT_TABLE-BKTXT.


      LOOP AT IT_TABLE ASSIGNING <FS_TABLE>.
        CLEAR WA_TABLE1.
        READ TABLE LT_TABLE1 INTO WA_TABLE1 WITH KEY EX_POSNUM = <FS_TABLE>-BKTXT.
        IF SY-SUBRC =  0.
          <FS_TABLE>-VBELN = WA_TABLE1-VBELN.
          <FS_TABLE>-ERDAT = WA_TABLE1-ERDAT.
          ZVBELN2          = WA_TABLE1-VBELN.
        ENDIF.
      ENDLOOP.

      SELECT SINGLE BELNR INTO ZBELNR FROM BKPF
        WHERE AWKEY = ZVBELN2
        AND AWTYP = 'VBRK'.

      LOOP AT IT_TABLE ASSIGNING <FS_TABLE>.
        <FS_TABLE>-BELNR2 = ZBELNR.
      ENDLOOP.

    ENDIF.

    IF LT_TABLE3 IS NOT INITIAL.
      CLEAR WA_TABLE3.
      LOOP AT LT_TABLE3  INTO WA_TABLE3 .
        CLEAR WA_TABLE.
        IF WA_TABLE3-AWTYP = 'VBRK' .
          WA_TABLE-VBELN  = WA_TABLE3-AWKEY.
          WA_TABLE-BELNR2 = WA_TABLE3-BELNR.
          WA_TABLE-BUKRS  = WA_TABLE3-BUKRS.
          WA_TABLE-GJAHR  = WA_TABLE3-GJAHR.

FORM GET_DATE."联营商品 商品凭证 对应的 成本会计凭证 在收入的会计凭证中体现。
*  故BKPF表中 不会记录联营商品的商品凭证号，不存在以下关系。
*  商品凭证与会计凭证的关系 BKPF-AWTYP = 'MKPF'
*                         BKPF-AWKEY = 商品凭证号码（MKPF-MBLNR）+商品凭证年度（MJAHR）

  DATA LT_MKPF TYPE STANDARD TABLE OF STRU_0010.
  DATA LW_MKPF TYPE                   STRU_0010.
  DATA LT_PASS TYPE STANDARD TABLE OF STRU_0020.

  MOVE-CORRESPONDING IT_TABLE TO LT_PASS.

  SELECT BKTXT
         MBLNR
         MJAHR
    FROM  MKPF
    INTO CORRESPONDING FIELDS OF TABLE LT_MKPF
    FOR ALL ENTRIES IN LT_PASS
    WHERE BKTXT = LT_PASS-BKTXT.

  LOOP AT LT_MKPF INTO LW_MKPF.
    CLEAR WA_TABLE.
    READ TABLE IT_TABLE INTO WA_TABLE WITH KEY MBLNR = LW_MKPF-MBLNR
                                                      MJAHR = LW_MKPF-MJAHR
                                                      BKTXT = LW_MKPF-BKTXT.
    IF SY-SUBRC <> 0.
      WA_TABLE-MBLNR = LW_MKPF-MBLNR.
      WA_TABLE-MJAHR = LW_MKPF-MJAHR.
      WA_TABLE-BKTXT = LW_MKPF-BKTXT.
      APPEND WA_TABLE TO IT_TABLE.
    ENDIF.
  ENDLOOP.

  SORT IT_TABLE BY BKTXT.

  DATA: LT_PASS2 TYPE STANDARD TABLE OF STRU_0010,
        LW_PASS2 TYPE                   STRU_0010.

  LOOP AT IT_TABLE ASSIGNING <FS_TABLE>.
    IF <FS_TABLE>-IDOC IS NOT INITIAL AND <FS_TABLE>-VBELN IS NOT INITIAL.
      CLEAR LW_PASS2.
      LW_PASS2 = <FS_TABLE>.
    ELSE.
      <FS_TABLE>-IDOC   = LW_PASS2-IDOC.
      <FS_TABLE>-VBELN  = LW_PASS2-VBELN.
      <FS_TABLE>-BELNR2 = LW_PASS2-BELNR2.
      <FS_TABLE>-BUKRS  = LW_PASS2-BUKRS."公司代码
      <FS_TABLE>-GJAHR  = LW_PASS2-GJAHR." 会计年度
      <FS_TABLE>-ERDAT  = LW_PASS2-ERDAT.
      <FS_TABLE>-TEST   = '联营商品成本结转信息，详见收入凭证。'."联营商品的成本，记在收入会计凭证中
      CLEAR LW_PASS2.
    ENDIF.
  ENDLOOP.
